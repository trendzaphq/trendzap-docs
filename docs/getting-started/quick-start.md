# Quick Start

Get started with TrendZap in just 5 minutes.

## Prerequisites

- A web browser (Chrome, Firefox, Brave)
- A compatible wallet (MetaMask, Core Wallet, Coinbase Wallet, or WalletConnect)
- Native tokens for gas fees
- USDC for betting

## Network Configuration

<NetworkConfig network="testnet" />

## Step 1: Connect Your Wallet

1. Visit [trendzap.xyz](https://trendzap.xyz)
2. Click **Connect Wallet** in the top right
3. Select your wallet provider
4. Approve the connection

:::tip Adding the Network
If the network isn't in your wallet yet, use the configuration above to add it manually, or let TrendZap add it automatically when you connect.
:::

## Step 2: Get Native Tokens

You need native tokens for gas fees. Options:

### Get Testnet Tokens
Use the faucet link in the network configuration above to get free testnet tokens.

### Bridge (Mainnet)
Bridge tokens from other networks using official bridges.

### Buy Directly
Purchase tokens on major exchanges and withdraw to the network.

## Step 3: Get USDC

TrendZap uses USDC for betting. Get it by:

1. **Faucet** - Get testnet USDC from the faucet (testnet only)
2. **Swapping** - Swap native tokens for USDC on a DEX
3. **Bridging** - Bridge USDC from another network

## Step 4: Place Your First Bet

1. **Browse Markets** - See active prediction markets
2. **Pick a Market** - Click on one that interests you
3. **Choose Side** - Select OVER or UNDER
4. **Enter Amount** - How much USDC to bet
5. **Confirm** - Approve the transaction in your wallet

🎉 **Congratulations!** You've placed your first TrendZap bet!
