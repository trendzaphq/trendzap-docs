# SDK Reference

:::caution Coming Soon
The TrendZap SDK is currently in development. This page will be updated when the SDK is available for public use.
:::

## What to Expect

The TrendZap SDK will provide a first-class TypeScript/JavaScript client for interacting with TrendZap markets — no need to write contract calls manually.

**Planned capabilities:**

- Browse and fetch market data
- Place bets programmatically
- Listen for market resolution events
- Create markets via API
- Retrieve user positions and payout history
