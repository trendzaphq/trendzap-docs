# Subgraph

:::caution Coming Soon
The TrendZap subgraph is under development. This page will be updated when it is live and queryable.
:::

## What to Expect

The TrendZap subgraph indexes all on-chain market events from Avalanche C-Chain and exposes them via a GraphQL API — enabling fast, filterable queries without reading directly from the chain.

**Planned entity coverage:**

- Markets (all parameters, status, outcome)
- Bets (per market, per user)
- Resolution events
- Claim events
- Volume and liquidity aggregates
