# Why Arbitrum?

TrendZap is built exclusively on **Arbitrum** - and we're committed to staying.

## Our Choice

After evaluating multiple chains, we chose Arbitrum as TrendZap's permanent home:

### ⚡ Speed & Cost

| Metric | Arbitrum | Ethereum |
|--------|----------|----------|
| Tx Finality | ~2 seconds | ~13 seconds |
| Avg Gas Cost | $0.01-0.10 | $5-50+ |
| TPS | ~40,000 | ~15 |

For a prediction market where users make frequent small bets, low fees are essential.

### 🔐 Security

Arbitrum inherits Ethereum's security:
- All transactions settle to Ethereum L1
- Fraud proofs protect against invalid state
- Battle-tested since August 2021

### 🌐 Ecosystem

Arbitrum has the largest L2 ecosystem:
- $3B+ TVL
- 500+ deployed protocols
- Strong DeFi primitives (Uniswap, Aave, GMX)
- Active developer community

### 🛠️ Developer Experience

- Full EVM compatibility
- Familiar tooling (Hardhat, Foundry)
- Comprehensive documentation
- Active grants program

## Not Multichain

We're **not** going multichain. Here's why:

1. **Focus** - Building on one chain lets us optimize deeply
2. **Liquidity** - Fragmented liquidity hurts prediction markets
3. **Complexity** - Cross-chain bridges add attack surface
4. **Community** - One community is stronger than many small ones

## Arbitrum Grant Program

We're actively participating in the Arbitrum ecosystem:

- Building with Arbitrum-native tools
- Contributing to Arbitrum developer resources
- Engaging with the Arbitrum community
